## Scripts
- `fasta_summary.sh` - Summarizes any FASTA file (sequence count, lengths, total bases)
- `cancer_report.sh` - Analyzes multiple cancer FASTA files, identifies biomarkers
- `count_nucleotides.sh` - Counts A, C, G, T in any DNA sequence
- `revcomp.sh` - Generates reverse complement of any DNA sequence
